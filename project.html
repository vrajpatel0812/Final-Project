Weather App


The Weather App displays current weather conditions for a given city, including temperature, humidity, wind speed, and a weather description (e.g., sunny, cloudy, rainy). It fetches weather data from a public API like OpenWeatherMap, and the user can search for any city to get the forecast. It could also show a 7-day forecast, or details like sunrise, sunset, and the pressure of the atmosphere.
